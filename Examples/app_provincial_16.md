# 第十六届示例说明

对应代码：[app_provincial_16.c](D:/桌面/2023蓝桥杯省国赛训练过程/蓝桥杯单片机通用模板工程/Examples/app_provincial_16.c)

## 这次为什么重写

前一版第十六届示例是基于公开复盘整理出来的，主线是对的，但你这次补的题面截图把很多细节钉死了：

- 页面格式
- 按键定义
- 光强等级阈值
- 运动状态锁定机制
- 参数页输出锁定
- `S8+S9` 组合清零

所以我把第十六届又按题面图重写了一次，现在这版更贴题面，而不只是“复盘版思路”。

## 题目主干

从你给的题面图里，可以稳定确认这题是：

- 环境状态界面
- 运动检测界面
- 参数设置界面
- 统计数据界面

外设只有：

- 光敏电阻
- `DS18B20`
- 超声波
- `LED`
- 继电器
- 数码管
- 键盘

没有 `RTC`，也没有 `DAC`，所以这题本质上是“状态判断题”，不是“外设拼装题”。

## 我是怎么从当前模板改出来的

### 第 1 步：先把页面拆准

我先按题面直接拆成：

- `PAGE_ENV`
- `PAGE_MOVE`
- `PAGE_PARAM`
- `PAGE_STAT`

参数页里再拆成两个子参数：

- `PARAM_TEMP`
- `PARAM_DIST`

这样做的好处是：

- `S4` 只管四个主页面循环
- `S5` 只在参数页里切换 `PC / PL`

页面层次会非常清楚。

### 第 2 步：先按题目阈值定义光强等级

题面给的条件是：

- `N1`：`V_AIN1 >= 3.0V`
- `N2`：`2.0V <= V_AIN1 < 3.0V`
- `N3`：`0.5V <= V_AIN1 < 2.0V`
- `N4`：`V_AIN1 < 0.5V`

因为模板里 `ADC` 是 0~255 对应 0~5V，所以我直接把它换成原始值阈值：

- `3.0V -> 153`
- `2.0V -> 102`
- `0.5V -> 26`

也就是说，环境等级判断不是凭感觉分档，而是按题面电压阈值来分。

### 第 3 步：再按题面写运动状态

题面要求：

- 每隔 1 秒采一次距离
- 用连续两次距离差 `ΔL` 判断运动状态
- `L1`：静止，`ΔL < 5cm`
- `L2`：徘徊，`5cm <= ΔL < 10cm`
- `L3`：跑动，`ΔL >= 10cm`

所以我专门把这块拆成：

- `g_last_distance`
- `g_delta_distance`
- `g_motion_state`
- `App_UpdateMotionState()`

这样运动状态判断会独立、清楚，而且后面改阈值也方便。

### 第 4 步：把状态锁定单独做出来

题面图里最关键的一条是：

- 当运动状态变化后，要立即更新
- 然后锁定 3 秒
- 锁定期间仍继续采样距离，但不再更新运动状态

这一点我没有塞进显示函数，而是单独做成：

- `g_motion_lock_ms`

在 `100ms` 节拍里倒计时，配合 1 秒采样节拍一起工作。

这一步是第十六届最像“国赛味”的地方，因为它不只是判断一次结果，而是管理状态生命周期。

### 第 5 步：把输出控制按题面拆开

题面要求：

- 触发“接近”时，`L1~L4` 根据光强等级点亮
- 未触发“接近”时，`L1~L4` 全灭
- `L8` 反映运动状态
  - 静止：灭
  - 徘徊：常亮
  - 跑动：0.1 秒闪烁
- 同时触发“接近”和“高温”时，继电器吸合

所以我把输出分成两块：

- `App_ApplyLedOutput()`
- `App_ApplyRelayOutput()`

这样灯和继电器不会混在一起写。

### 第 6 步：参数页加入“输出锁定”

题面明确要求：

- 进入参数界面时，`LED` 和继电器状态锁定，不可变化
- 退出参数界面后，取消锁定

所以我加了：

- `g_output_locked`
- `App_EnterParamPage()`
- `App_ExitParamPage()`

也就是说，传感器采样还能继续跑，但在参数页里不把新的状态刷新到灯和继电器上。

### 第 7 步：统计页补组合键清零

题面要求：

- 在统计页下
- `S8`、`S9` 同时按住超过 2 秒
- 清零继电器吸合次数

模板原本只有单键事件接口，所以我额外补了：

- `App_IsKey8And9PressedNow()`

然后在 `100ms` 节拍里自己做组合键长按计时。

这一步很有代表性，因为它说明：

- 模板不一定一开始就包含真题全部动作
- 但可以在业务层里补一小段“题目专用输入逻辑”

### 第 8 步：最后才按题图摆数码管

等业务和状态都定下来后，我再按图做显示：

- 环境页：`C23   N4`
- 运动页：`L1   035`
- 参数页：`PC   32` 或 `PL   40`
- 统计页：`NC   42`

注意这里我还顺手补了一个字符：

- `SEG_L`

因为第十六届页面里 `L1 / PL` 都需要直接显示 `L`。

## 这版和前一版第十六届相比，最重要的变化

- 运动页左侧从近似写法改成题面要求的 `L1/L2/L3`
- 参数页从通用写法改成题面要求的 `PC / PL`
- 按题面补了参数页输出锁定
- 按题面补了 `S8+S9` 组合长按清零
- 按题面修正了光强等级阈值

## 看代码时最值得先看的地方

推荐顺序：

1. `App_UpdateLightLevel()`
2. `App_UpdateMotionState()`
3. `App_EnterParamPage()` / `App_ExitParamPage()`
4. `App_IsKey8And9PressedNow()`
5. `App_Loop()`

这 5 块基本就是第十六届最核心的 5 个难点：

- 模拟量分级
- 连续采样状态判断
- 页面切换和输出锁定
- 组合键长按
- 最后如何挂回稳定模板的调度系统
