# 第十六届示例说明

对应代码：[app_provincial_16.c](./app_provincial_16.c)

## 这题主要练什么

- 环境状态判断
- 光照等级判定
- 距离接近与运动状态判断
- 参数页与统计页
- LED / 继电器联动

## 先改哪 5 处

### 1. 先拆四个主页面

优先按题面直接拆成：

- 环境页
- 运动页
- 参数页
- 统计页

### 2. 再定状态量

这题核心不是外设多，而是状态多。  
建议先定清楚：

- 光照等级
- 当前距离
- 接近状态
- 运动状态
- 统计次数

### 3. 再接光照、温度、超声波

建议顺序：

1. 光照等级
2. 温度
3. 距离与运动状态

### 4. 再写状态锁定和参数逻辑

这题最关键的是“运动状态变化后锁定 3 秒”，一定要单独处理。

### 5. 最后补 LED / 继电器 / 统计清零

输出和统计都依赖前面的状态判断已经正确。

## 这题最容易卡住的地方

- 把“接近判断”和“运动状态判断”混成一件事
- 忘记状态锁定
- 参数页时输出没有锁定
- 统计清零没有做成长按或组合键判断

## 建议验证顺序

1. 先验证四页面切换
2. 再验证光照等级显示
3. 再验证距离和运动状态
4. 再验证参数修改
5. 最后验证 LED、继电器和统计页
