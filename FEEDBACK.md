# Feedback Guide

如果你在使用这套工程时发现问题，建议按下面格式反馈。

## 反馈范围

欢迎反馈以下内容：

- 编译报错
- 数码管显示异常
- 按键功能异常
- 温度、超声波、频率、时钟等数据异常
- 示例代码与官方题面不一致
- 文档表述不清楚

## 建议反馈格式

### 1. 说明问题发生位置

- 主模板工程
- 某一届 `Examples`
- 某个具体模块

### 2. 说明题目或文件

请尽量写清楚：

- 第几届题目
- 哪个文件
- 哪个页面或功能

### 3. 提供现象

例如：

- 编译器报了什么错
- 上板后显示成什么
- 按下哪个键没有反应
- 传感器读数为什么不对

### 4. 提供你已经验证过的信息

例如：

- 是否能正常编译
- 是否已经烧录成功
- 是否更换过 `App/app.c`
- 是否核对过题面截图
- 是否只有某一届示例有问题

## 推荐反馈示例

```text
问题位置：Examples/app_provincial_10.c
题目届数：第十届省赛
问题现象：替换进 App/app.c 后可以编译，但频率页面显示和题面不一致
已验证信息：主模板本身可以正常运行；我已经上板；数码管和按键正常
补充说明：题面要求只有频率页、参数页、时间页和回显页
```

## 维护原则

这套仓库按“边使用、边验证、边修正”的方式维护：

- 主模板优先保证稳定
- 示例代码优先保证题意迁移和学习价值
- 文档优先帮助零基础快速进入训练状态

如果反馈信息足够明确，后续就可以更高效地继续修订。
