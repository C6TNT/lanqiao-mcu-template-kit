# Changelog

## V1.0 - 2026-04-05

首个可公开维护版本。

### Added

- 建立统一的蓝桥杯单片机通用模板工程目录结构
- 补充 `App / BSP / Core / Devices / Drivers` 分层代码
- 补充数码管、按键、串口、温度、时钟、超声波、频率、ADC/DAC 等常见竞赛模块
- 补充双击、长按、短按按键接口
- 补充 `Docs` 文档体系
- 补充 `Examples` 历届省赛模板版示例及配套说明
- 新增仓库根目录 `README.md`
- 新增 `.gitignore`
- 新增 `VERSION`
- 新增反馈说明文档

### Changed

- 清理仓库结构，只保留和蓝桥杯单片机比赛相关的核心资料
- 明确 `Examples` 的定位为学习版、复盘版、模板迁移版
- 强化主模板作为稳定训练工程的定位
- 按“零基础可读性”思路重写默认 `App/app.c` 的注释结构
- 新增模板源码导读和赛题要求对照文档
- 统一重写 `Examples` 下各届 `.md` 说明结构
- 新增“新手最短阅读入口”文档
- 新增编译报错支持文档三件套

### Notes

- 当前主模板工程适合直接作为训练和刷题起点使用
- `Examples` 建议边学习边编译、边上板、边核对题面
- 第九届、第十届、第十一届示例尤其建议在使用时同步验证
